# **App Name**: ConstructConnect

## Core Features:

- Product Showcase: A visually appealing main page to highlight key construction materials and promotional offers.
- User Registration (Normal): Registration form for individual users, collecting Name, Last Name, Phone Number, Email, and a secure Password (minimum 8 characters with numbers, uppercase, lowercase, and a special symbol).
- User Registration (Business): Registration form for business users, collecting Company Name, Legal Representative Name, Email, RFC (Tax ID), Phone Number, and a secure Password (minimum 8 characters with numbers, uppercase, lowercase, and a special symbol).
- User Authentication: Secure login functionality for both normal and business users, verifying credentials against stored user data.
- Local Testing Account: A pre-configured user account to facilitate local testing and development.

## Style Guidelines:

- Primary color: Earthy terracotta (#E07A5F) evoking the raw materials of construction.
- Background color: Light beige (#F8F0E3), a desaturated tint of terracotta for a clean backdrop.
- Accent color: Muted teal (#3D405B) to provide contrast and a professional feel.
- Body and headline font: 'PT Sans', a modern sans-serif combining warmth and legibility for clear communication.
- Use solid, geometric icons representing construction materials and user actions.
- Clean and structured layout with clear sections for product display and user registration.
- Subtle transition animations on button hover states to indicate interactivity.
